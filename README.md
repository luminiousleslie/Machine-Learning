# Machine-Learning-Models
###Welcome to my Machine Learning Repository ^_^! This project is a collection of Machine Learning (ML) models implemented using libraries like PyTorch, Scikit-Learn, PyTorch. The repository is organized based on the following learning categories: 
        *supervised: for models trained with labeled data
        *unsupervised: for models trained to with unlabeled data
        *semi-supervised: for models trained with a blend of labeled and unlabeled data
